# project_sprint7

# Aplicación Web de Análisis de Vehículos
# Descripción

Esta aplicación web permite analizar datos de vehículos usados mediante visualizaciones interactivas. Los usuarios pueden explorar la relación entre precio, kilometraje, condición y otras características de los vehículos a través de histogramas y gráficos de dispersión.

# Instrucciones de instalación
1. Crear entorno virtual: `conda create -n vehicles_env python=3.9`
2. Activar entorno: `conda activate vehicles_env`
3. Instalar librerías: `pip install streamlit pandas plotly-express`
